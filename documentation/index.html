Flutter Setup

You can present your app features. It is all very easy!

Open Exsiting Project:

   • Step 1:   Start Vs Code and Open Exsiting Project Select Project 

flip_card_memory_game





  • Step 2:   :    Run the app :

To run the app from the command line:

flutter run




   • Step 3:    Change the app package name:

https://pub.dev/packages/change_app_package_name 

flutter pub run change_app_package_name:main com.new.package.name




   • Step 4:    Change the app name:

https://pub.dev/packages/rename_app 

flutter pub run rename_app:main all="My App Name"




   • Step 5:    Change the app icon:

https://pub.dev/packages/flutter_launcher_icons

Edit flutter_launcher_icons.yaml with your content

Run the following commnd: 

flutter pub run flutter_launcher_icons -f flutter_launcher_icons.yaml




   • Step 6:    Change the splash screen:

https://pub.dev/packages/flutter_native_splash

Edit flutter_native_splash.yaml with your content

Run the following commnd: 

dart run flutter_native_splash:create --path=flutter_native_splash.yaml





   • Step 7:   Add your Admob Keys:

Add your Admob Keys in config.dart file :

flutter_app/lib/helpers/config.dart

flutter_app/android/app/src/main/AndroidManifest.xml
